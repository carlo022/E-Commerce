# 🛍️ E-Commerce Platform

## Overview
A **full-stack e-commerce application** built with modern web technologies, featuring a responsive user interface and RESTful backend API. This project demonstrates expertise in building scalable, production-ready web applications with React, Node.js, and JSON-Server authentication.

**Live Demo**: [eshopus.netlify.app](https://eshopus.netlify.app/)

---

## ✨ Key Features

### Frontend
- **React 19** - Modern UI framework with hooks and latest features
- **React Router v6** - Client-side routing for seamless navigation
- **Tailwind CSS** - Utility-first CSS framework for responsive design
- **Toast Notifications** - User feedback with react-toastify
- **Vite** - Lightning-fast build tool for optimal development experience

### Backend
- **JSON-Server with Authentication** - Mock REST API with JWT authentication
- **Express.js** - Flexible and scalable backend framework
- **Custom Routes** - API customization through routes configuration
- **Database** - JSON-based persistent data storage

### Development Tools
- **ESLint** - Code quality and consistency
- **Netlify Deployment** - Automated CI/CD pipeline
- **Modular Architecture** - Clean separation of concerns

## 🌐 API Endpoints

The backend provides RESTful API endpoints with JWT authentication:

```
/api/products      - Product management
/api/cart          - Shopping cart operations
/api/orders        - Order management
/api/users         - User authentication
```

For detailed API documentation, refer to `backend/data/routes.json`
